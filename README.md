# meteor-todo
Meteor Todo App

### installing curl
$ sudo apt install curl

### instaling meteor
$ curl https://install.meteor.com/ | sh

### clone repository
$ git clone https://github.com/martino1988/meteor-todo.git

### change directory
$ cd meteor-todo

### start app
$ meteor npm install

$ meteor
